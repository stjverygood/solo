## 玩法描述

```
玩家开局会出生在一个随机的世界, 作为凡人;
然后开始去撸树和撸石头, 捡垃圾;
收集好材料后开始到处逛;
勘察附近修仙势力/农田情况/山洞的情况, 并寻找首个落地的地方;
好的落地环境 : 附近比较少强大修仙势力(修仙势力会威胁玩家活动), 有丰富农田(可以种灵药), 山洞也比较丰富(探险, 刷装备)
落地后需要抵御晚上的妖兽袭击, 并整理白天获取的物质, 开始种种田;
白天继续找资源/进洞穴冒险/进深海/

目标 : 通过采集/探险/攻城, 不断提升修为, 直至升仙

倒推 : 
提升修为能解锁科技, 解锁新产能方法(如点金术/生铁术), 提高基础物资产量
提升修为的核心道具 : 
	1. 长春功/聚灵丹/洗髓丹(进入炼气期); 
	2. 筑基丹(进入筑基期); 
	3. 天雷劫(强大boss妖兽的袭击, 进入金丹期) 
	4. 定灵丹/九曲灵参/定神香/破魔丹 + 大天劫(凑齐道具后召唤渡劫仪式, 渡劫后进入元婴期)
	5. 杀死其余全部修仙家族后, 经历化凡, 各种一系列磨练, 最后化神
进阶机制 : 进阶条件满足得越多, 成功进阶的概率越大(最高100%), 但永远有几个条件很难满足, 失败后会吃掉条件, 让玩家要么赌, 要么乖乖刷齐条件

物品 : 
	1. 工具类 : 采集(稿/斧) + 种田(锄/壶) + 战斗(剑/弓/头/身/脚)
	2. 材料类 : 食物, 合成材料(工具/建筑), 种子, 进阶道具(各种丹/秘籍), 装备强化材料(符文/器灵)
	3. 摆放类 : 建筑
工具等级 : 木/石/铁/金
采集材料 : 
	1. 树 : 只有斧头能挖, 其他工具只-1, 掉落木头
	2. 石头 : 木镐以上能挖, 掉落石头
	3. 铁矿石 : 稀少, 石镐以上能挖, 掉落铁矿石, 需要在火炉里炼制成铁才能使用
	4. 金矿石 : 极其稀少, 铁镐以上能挖, 只在特殊地方刷新, 需要火炉炼制成金, 金后续可注入灵气升级, 合成更强大装备 
	5. 其他各种能直接捡起来的东西

建筑 : 
	1. 阵法之眼 : 木头 + 灵气之源(生成在出生点附近, 后期非常偶尔能获取)
	2. 阵旗 : 木头, 拓展灵气范围
	3. 火炉 : 炼制矿石, 需要补充燃料
	4. 炼丹炉 : 把材料练成丹药, 补充灵气, 提高修为
	5. 储物箱 : 存放物品
	6. 建筑制作台
	7. 

玩家 : 
属性 : 
	1. 血量 : 降到0会死亡, 死亡后会删档
	2. 灵气(饱食度) : 满灵气后会自动回血, 为0时不再恢复体力值, 靠吃东西增长
	3. 体力 : 冲刺, 闪避消耗, 能自动恢复
	4. 修为 : 经验, 进阶的条件, 进阶失败后会从头来过
```
