## 玩法描述

```
玩家开局会出生在一个随机的世界, 作为凡人;
然后开始去撸树和撸石头, 捡垃圾;
收集好材料后开始到处逛;
勘察附近修仙势力/农田情况/山洞的情况, 并寻找首个落地的地方;
好的落地环境 : 附近比较少强大修仙势力(修仙势力会威胁玩家活动), 有丰富农田(可以种灵药), 山洞也比较丰富(探险, 刷装备)
落地后需要抵御晚上的妖兽袭击, 并整理白天获取的物质, 开始种种田;
白天继续找资源/进洞穴冒险/进深海/

目标 : 提升身上装备 + 提升修为 -> 把全图的修仙势力干掉

倒推 : 
提升修为能解锁科技, 解锁新产能方法(如点金术/生铁术), 提高基础物资产量
提升修为的核心道具 : 
	1. 长春功/聚灵丹/洗髓丹(进入炼气期); 
	2. 筑基丹(进入筑基期); 
	3. 天雷劫(强大boss妖兽的袭击, 进入金丹期) 
	4. 定灵丹/九曲灵参/定神香/破魔丹 + 大天劫(凑齐道具后召唤渡劫仪式, 渡劫后进入元婴期)
	5. 杀死其余全部修仙家族后, 经历化凡, 各种一系列磨练, 最后化神
进阶机制 : 进阶条件满足得越多, 成功进阶的概率越大(最高100%), 但永远有几个条件很难满足, 失败后会吃掉条件, 让玩家要么赌, 要么乖乖刷齐条件

装备 : 
	1. 工具类 : 采集(稿/斧) + 种田(锄/壶) + 战斗(剑/弓/头/身/脚)
	2. 材料类 : 食物, 合成材料(工具/建筑), 种子, 进阶道具(各种丹/秘籍), 装备强化材料(符文/器灵)
工具等级 : 木/石/铁/金
采集材料 : 树(只有斧头能攻击, 其他工具只-1)/石(镐子)/铁矿石(稀少)/金矿石(只在特殊地方刷新, 及其稀少), 各种能直接捡起来的东西

```
